# Banking Operational Error Detection

project analyzes operational transaction errors in banking systems.

Objectives:
- Detect transaction failures
- Identify SLA violations
- Detect fraud
- Provide root cause insights

Technologies:
Python (Pandas, Matplotlib, Seaborn)
SQL (SQLite / PostgreSQL compatible queries)
Power BI for business dashboards
Jupyter Notebook for analysis
GitHub for version control

Dataset:
PaySim Financial Dataset (Kaggle)
